# [CLUE] Redox — Identifying Oxidizing & Reducing Agents
> Cover everything below and lecture it from memory.

- **Core mnemonics**
  - **OIL RIG**: Oxidation Is Loss (of electrons) · Reduction Is Gain (of electrons)
  - **LEO the lion says GER**: Loses Electrons = Oxidized · Gains Electrons = Reduced

- **Agent definitions** (the agents are the *other* species — what they DO to themselves)
  - **Oxidizing agent** = gets **reduced** (gains e⁻); causes the other species to be oxidized
  - **Reducing agent** = gets **oxidized** (loses e⁻); causes the other species to be reduced
  - Note: "The oxidizing agent is the one being reduced" — sounds backwards, memorize it cold

- **How to identify what's oxidized vs. reduced: assign oxidation numbers**
  - Rules (apply in order):
    1. Pure element → **0**
    2. Monatomic ion → **its charge** (Na⁺ = +1, Cl⁻ = −1)
    3. **O = −2** (except peroxides O₂²⁻: O = −1; OF₂: O = +2)
    4. **H = +1** (except metal hydrides, e.g. NaH: H = −1)
    5. Sum of oxidation numbers = **overall charge** of the species (0 if neutral)
  - Once you have oxidation numbers for each element across the reaction:
    - Number **increases** → that element is **oxidized** → its species is the **reducing agent**
    - Number **decreases** → that element is **reduced** → its species is the **oxidizing agent**

- **Worked example: Zn + CuSO₄ → ZnSO₄ + Cu**
  - Zn: 0 → +2 (increases) → **oxidized** → Zn is the **reducing agent**
  - Cu: +2 → 0 (decreases) → **reduced** → Cu²⁺ is the **oxidizing agent**

- **Quick pattern recognition**
  - Strong oxidizing agents: F₂, Cl₂, O₂, H₂O₂, MnO₄⁻, Cr₂O₇²⁻, concentrated H₂SO₄/HNO₃
  - Strong reducing agents: alkali metals (Na, Li), H₂, C (in high-temp reactions)
  - Combustion: fuel = reducing agent; O₂ = oxidizing agent (always)
