# [CLUE] Enzyme Function, Naming, and Structure
> Cover everything below and lecture it from memory.

- **Enzyme = catalyst**
  - speeds up reactions without being used up
  - structure determines function
    - Note: something binds the **allosteric site** → changes enzyme shape → active site altered → function changes
    - Note: **mutation** → changes DNA seq → changes RNA seq → changes AA seq (unless silent) → changes enzyme structure → function changes

- **Naming** — based on reaction type, ends in "-ase"
  - **Oxidase** = oxidizes something else (removes e⁻ from that thing)
    - **Dehydrogenase** = a type of oxidase
      - Note: removes a **hydride** (NOT hydrogen) to remove e⁻
    - Note: covalent bonds are made of electrons; removing e⁻ = fewer bonds → oxidation breaks things down (rust from Fe, CO₂ from glucose)
  - **Reductase** = reduces something else (adds e⁻ to that thing)
    - Note: **OIL RIG** — Oxidation Is Loss, Reduction Is Gain
    - Note: adding e⁻ = more bonds → reduction builds larger molecules (reductive synthesis)
  - **Kinase** = adds a phosphate group
    - Note: "kin" (kinship) = together → it ADDS phosphate
  - **Phosphatase** = removes a phosphate group
  - **Hydrolases** = break bonds via hydrolysis
    - Note: "hydro" = water; they add water to split molecules
    - **Protease** = breaks down proteins (hints "protein")
    - **Amylase** = breaks down carbohydrates
    - **Lipase** = breaks down fats (hints "lipid")
    - **Nucleases** = hydrolyze phosphodiester bonds (hints "nucleus")
      - **DNAse** = breaks DNA · **RNAse** = breaks RNA
  - **Polymerase** = builds polymers (has "polymer" in it)
    - **DNA polymerase** = builds DNA from nucleotides
    - **RNA polymerase** = builds RNA from nucleotides
  - **Ribosome** = builds proteins from amino acids
  - **Transcriptase** = makes RNA from a DNA template
  - **Reverse transcriptase** = makes DNA from an RNA template
    - Note: reverses transcription — RNA template → DNA
  - **Ligase** = joins DNA or RNA strands
    - Note: a "ligature" is a string that joins things

- **Specificity**
  - binding happens at the **active site**
    - Note: the reACTION occurs at the ACTIVE site
  - structural fit creates specificity
    - Note: if enzymes weren't specific they'd catalyze random reactions and wreck everything
  - can distinguish **stereoisomers** → enzymes are stereospecific
    - Note: wrong chirality won't fit — like a left hand in a right-handed glove
  - **Models**
    - **Lock & key** = rigid active site, exact fit (key only works on exact match)
    - **Induced fit** = flexible active site molds to grip the substrate (like play-doh)

- **Local conditions & activity** — optimal conditions needed for peak activity
  - **Temperature** — Note: cooking an enzyme ruins it (denatures)
  - **pH** — Note: pouring acid/base on it ruins it
  - **Salt concentration** — Note: tertiary structure often relies on ionic bonds between side chains; salt ions get in the way

- **Structure**
  - most enzymes are **protein**
    - Note: the ribosome is mostly **RNA** (a **ribozyme**)
    - Note: early life was likely RNA-based — the first ribosome was RNA before it made proteins
  - **Levels of structure** (determine function)
    - **Primary** = sequence
      - proteins: AA sequence · nucleic acids: nucleotide sequence
    - **Secondary** = due to H-bonding
      - proteins: α-helices & β-sheets · nucleic acids: base pairing (H-bonds)
    - **Tertiary** = 3D arrangement from side-chain interactions
      - H-bonding · hydrophobic interactions · ionic bonds · **disulfide bridges**
        - Note: two cysteines → a cystine; this is a **covalent** bond → very strong
    - **Quaternary** = multiple chains combine
      - dimer = 2 · trimer = 3 · tetramer = 4 · **oligomer = 2+**
        - Note: so a dimer is also an oligomer
