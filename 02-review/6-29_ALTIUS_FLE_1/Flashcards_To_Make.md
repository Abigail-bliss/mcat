## From: BBF_Passage_8.MD

- [x] **Protein structure levels — what each level IS and what disrupts it** — Primary = amino acid sequence (any non-synonymous point mutation changes it); Secondary = local backbone folding into α-helices/β-sheets (H-bonds between backbone atoms); Tertiary = overall 3D fold of one chain (R-group interactions: disulfide bonds, hydrophobic core, etc.); Quaternary = assembly of 2+ chains into a complex (dimers, trimers, oligomers). Key rule: any subunit assembly (trimer, dimer) = quaternary. A point mutation guarantees primary disruption; secondary/tertiary are NOT guaranteed — must be stated or implied. (From Q46.)

## From: BBF_Passage_7.MD

- [x] **Wobble hypothesis** — The 3rd codon position (wobble position) can pair with non-Watson-Crick bases in the tRNA anticodon, so one tRNA can recognize multiple codons for the same amino acid. Explains why fewer tRNAs than codons exist (64 codons, ~45 tRNAs). G can wobble-pair with U; inosine (I) can pair with U, C, or A.
- [x] **Hardy-Weinberg principle + equation** — Single dense card. Equations: p + q = 1 and p² + 2pq + q² = 1 (p = dominant allele freq, q = recessive). Punnett square showing p²(AA) + 2pq(Aa) + q²(aa). Five assumptions for H-W equilibrium (no mutation, no selection, no gene flow, random mating, large population). MCAT use: given q² (homozygous recessive phenotype freq), solve for q → p → carriers (2pq).

## From: CPF_Discrete_Questions.MD — Q44 (tryptophan intrinsic fluorescence)

- [x] **Intrinsic tryptophan fluorescence as a folding probe** — Trp's indole ring absorbs UV (~280 nm) and emits at a longer wavelength; emission intensity/wavelength depends on local environment (buried in hydrophobic core = brighter, blue-shifted; solvent-exposed = quenched, red-shifted). As a protein folds, Trp residues bury → fluorescence intensity rises over time → fit to a first-order rate to get the folding rate constant. It's a passive *readout* (reports environment), NOT a folding participant/chaperone, and does NOT itself "provide" secondary structure. ## From: self-study (not this exam) — Biochem/Bio

- [x] **Monomer / dimer / trimer / polymer** — **Monomer** = single subunit (e.g. one amino acid, one glucose). **Dimer** = 2 subunits (e.g. hemoglobin has 2α + 2β = tetramer, but many enzymes are homodimers). **Trimer** = 3 subunits. **Polymer** = many repeating monomers linked together (e.g. proteins, DNA, polysaccharides). Key MCAT rule: any multi-subunit assembly = **quaternary structure**; dimer/trimer/tetramer all count.

## From: self-study (not this exam) — Gen Chem

- [x] **Oxidizing vs. reducing agents + how to identify them** — OIL RIG: Oxidation Is Loss, Reduction Is Gain. Oxidizing agent = gets reduced (gains e⁻). Reducing agent = gets oxidized (loses e⁻). To identify: assign oxidation numbers → element whose number **increases** = oxidized = its species is the reducing agent; number **decreases** = reduced = its species is the oxidizing agent. Oxidation number rules: pure element = 0; monatomic ion = its charge; O = −2 (−1 in peroxides); H = +1 (−1 in metal hydrides); numbers must sum to species charge.

## From: self-study (not this exam) — Biochem

- [x] **Positive vs. negative cooperativity** — Cooperativity = binding of one substrate changes affinity for the next. **Positive cooperativity:** first binding event increases affinity for subsequent substrates (sigmoidal V vs. [S] curve, not hyperbolic). **Negative cooperativity:** first binding decreases affinity for subsequent substrates. Classic example: hemoglobin (positive cooperativity — O₂ binding makes next O₂ bind easier). Hill coefficient (n): n > 1 = positive; n < 1 = negative; n = 1 = no cooperativity (Michaelis-Menten). Sigmoidal curve = cooperativity on MCAT.

## From: self-study (not this exam) — Orgo

- [x] **Keto-enol tautomerism — which tautomer dominates & why** — Keto ⇌ enol: the **alpha H** (on the carbon adjacent to the carbonyl C) moves to the O (C=O → C–OH), and the double bond shifts to between the carbonyl C and the alpha C (C=O → C=C). "Enol" = **en** (alkene C=C) + **ol** (OH). Keto normally dominates at equilibrium. **Exceptions where enol wins (MCAT traps):** phenol (enol form is aromatic → stays enol) and 1,3-dicarbonyls (enol stabilized by conjugation + intramolecular H-bond).
